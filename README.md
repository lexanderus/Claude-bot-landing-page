# Claude TG Bot — Landing Page

Одностраничный сайт для Telegram-бота "Клот" — персонального AI-помощника с доступом к VPS и командой из 250+ специализированных субагентов.

## Как создать своего бота и подключить к VPS

### Шаг 1. Создайте бота
Откройте [@BotFather](https://t.me/BotFather) в Telegram, отправьте `/newbot`, введите имя и username (должен оканчиваться на `_bot`).

### Шаг 2. Получите токен
BotFather пришлёт токен вида `8725003802:AAGky…`. Сохраните его — это пароль от бота.

### Шаг 3. Подключите к VPS
Перешлите токен и свой `user_id` владельцу сервера (или в чат @KlotBot). Владелец:
- Создаёт системный юнит `systemd` для вашего тенанта
- Настраивает изолированную базу знаний (`pgvector-KB`)
- Выделяет песочницу с доступом к VPS-инструментам

### Шаг 4. Настройте модели
Выберите модели ИИ через `/model` в вашем новом боте. Можно использовать:
- 🔑 **Свой ключ** — подключите через `/setkey` (платите сами)
- 🤝 **Ключ мастера** — предоставляется владельцем (работает на балансе)

### Шаг 5. Готово!
Ваш персональный AI-ассистент работает на VPS. Делитесь ссылкой с командой.
