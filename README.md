# Клот — Landing Page

Одностраничный сайт для Telegram-бота **«Клот»** — персонального AI-ассистента с доступом к VPS и командой из 250+ специализированных субагентов.

## 🚀 Как создать своего бота

### Шаг 1. Создайте бота в @BotFather
Откройте [@BotFather](https://t.me/BotFather) в Telegram, отправьте `/newbot`.  
Введите имя (например «Мой ИИ-ассистент») и username (должен оканчиваться на `_bot`).

### Шаг 2. Получите токен
BotFather пришлёт токен вида `8725003802:AAGky…` — сохраните его, это пароль от бота.

### Шаг 3. Подключите к VPS
Перешлите токен и свой `user_id` владельцу сервера (или в чат [@VPSAndromedaBot](https://t.me/VPSAndromedaBot)).

### Шаг 4. Владелец настроит сервер
Создаст systemd-юнит, изолированную базу знаний (LightRAG) и подключит модели.

### Шаг 5. Выберите модели
Команда `/model` — выберите ИИ (GPT, Claude, DeepSeek, Gemini, OpenRouter). Бот готов!

## 📋 Команды бота

| Команда | Назначение |
|---------|-----------|
| `/start` | Приветствие |
| `/help` | Краткая справка |
| `/status` | Текущая модель и состояние |
| `/model` | Выбрать модель ИИ |
| `/profile` | Сменить провайдера |
| `/new` | Новый разговор (сброс контекста) |
| `/cancel` | Прервать текущую задачу |
| `/departments` | Список отделов-специалистов |
| `/kb` | Долговременная память |
| `/keys` | Статус ключей провайдеров |
| `/setkey` | Добавить/заменить свой ключ |
| `/delkey` | Убрать свой ключ |
| `/gcalauth` | Подключить Google Calendar |
| `/usage` | Расход: токены и стоимость |
| `/balance` | Баланс и пополнение |
| `/admin` | Панель владельца |
